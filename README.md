# Lighthouse Landing Page 🚢

AI-powered task decomposition tool landing page with waitlist functionality.

## Features

- 🌐 **Bilingual Support**: Auto-detects browser language (Korean/English)
- 🎨 **Generative Art**: p5.js-based animated background with particles and waves
- ✨ **Interactive Elements**: Spotlight cards with mouse-following effects
- 📧 **Waitlist System**: Automated thank-you emails via Resend
- 🎯 **Minimal Design**: Neutral palette with strategic amber accents

## Tech Stack

- **Frontend**: React 19 + Vite
- **Styling**: Tailwind CSS v3
- **Animation**: Framer Motion
- **Generative Art**: p5.js
- **Email**: Resend API
- **Deployment**: Vercel

## Setup

1. **Clone and install dependencies**
   ```bash
   npm install
   ```

2. **Configure email service**
   - Sign up at [Resend](https://resend.com)
   - Get your API key from [Dashboard → API Keys](https://resend.com/api-keys)
   - Verify your domain or use test mode

3. **Environment variables**
   ```bash
   cp .env.example .env
   ```
   
   Add your Resend API key to `.env`:
   ```
   RESEND_API_KEY=re_xxxxxxxxxxxxxxxxxxxxxxxxxxxxx
   ```

4. **Run development server**
   ```bash
   npm run dev
   ```

## Deployment (Vercel)

1. **Push to GitHub**
   ```bash
   git push origin main
   ```

2. **Connect to Vercel**
   - Import project from GitHub
   - Add environment variable: `RESEND_API_KEY`
   - Deploy

3. **Configure Resend domain (production)**
   - Go to Resend → Domains
   - Add your domain (e.g., `lighthouse.so`)
   - Update DNS records
   - Update `from` field in `api/waitlist.js`:
     ```javascript
     from: 'Lighthouse <hello@yourdomain.com>'
     ```

## Internationalization

The site automatically detects browser language and displays Korean or English content. Manual toggle available in header.

Supported languages:
- 🇺🇸 English
- 🇰🇷 한국어

Add more languages in `src/i18n.js`.

## Design Philosophy

**Minimal neutral tones with strategic accent color:**
- Base: Neutral grays (50-900)
- Accent: Amber (500) on hover/focus only
- Typography: Crimson Pro (serif) for headings
- Inspiration: Cursor, OpenAI, Anthropic design systems

## Email Templates

Two templates included in `api/waitlist.js`:
- English version
- Korean version

Both templates:
- Minimal design matching landing page
- Mobile-responsive
- Plain text fallback (auto-generated by Resend)
